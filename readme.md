# Uint RNG

A tiny insorphic Random Number Generator for generating 8/16/32-bits unsigned integers.

## Install

```sh
npm install --save uint-rng
```

## Usage

```ts
import RNG from 'uint-rng';

RNG.get8 (); // => 72
RNG.get16 (); // => 23291
RNG.get32 (); // => 1129029
```
